# Installation

1. Install the Tampermonkey browser extension: https://www.tampermonkey.net/
   *(I recommend using Firefox, since it also supports browser extensions on mobile.)*
2. Open the following URL in your browser: https://raw.githubusercontent.com/VRCUploader/vrcw-download-script/refs/heads/main/vrcwDownloader.user.js
   If Tampermonkey is installed correctly, it will automatically detect the userscript and display the installation page.
3. Click **Install**, and you're done!

# Usage

Once the script is installed, search for a world on **vrchat.com**. Each search result will now include a **Download VRCW** button.
